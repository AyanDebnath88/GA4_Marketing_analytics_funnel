# GA4 Digital Analytics Funnel

## Overview

This project simulates a real-world Google Analytics 4 implementation for an e-commerce company.

The objective is to build an end-to-end analytics solution beginning with GA4 event data, transforming it using Python, modeling it in SQL, and visualizing business insights in Power BI.

## Tech Stack

- Python
- SQL
- Power BI
- Google Analytics 4
- Git

## Business Goal

Help marketing teams answer:

- Which campaigns generate revenue?
- Which channels have the highest ROAS?
- Where do customers drop out of the funnel?
- Which landing pages convert best?
